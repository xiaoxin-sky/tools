    E: 'static,
    M: Display + 'static,
    Error: From<ContextDiagnostic<M, E>>, 

If self is an error, returns a new diagnostic with the provided message as a message and description, and self as a source diagnostic. This is useful to create chains of diagnostics, where high level errors wrap lower level causes.
